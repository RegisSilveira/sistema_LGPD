# Sistema LGPD

Sistema de gerenciamento e adequação à Lei Geral de Proteção de Dados (LGPD)

## Sobre o Projeto

Este projeto tem como objetivo auxiliar organizações a se adequarem e gerenciarem seus processos de acordo com a Lei Geral de Proteção de Dados (LGPD) - Lei nº 13.709/2018.

## Funcionalidades (Em desenvolvimento)

- Gestão de consentimento de dados
- Mapeamento de dados pessoais
- Registro de operações de tratamento
- Gestão de incidentes de segurança
- Relatórios de conformidade
